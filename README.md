# Агротех: Калькулятор полива теплицы

Учебный проект. Калькулятор для примерного расчёта объёма полива теплицы по площади, культуре, температуре и влажности почвы.

## Стек
- Python 3.7+
- Tkinter (стандартная библиотека)

git clone https://github.com/ВАШ_ЛОГИН/agro-calc.git
